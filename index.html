Play a track by name or URL. Supports YouTube, Spotify playlists, SoundCloud and direct file URLs. Auto-queues if something is already playing and shows playlist size for bulk imports.

Show a now-playing embed with track title, requester, a live progress bar and time remaining. Supports live streams with a 🔴 LIVE indicator.

Fetch lyrics for the currently playing track. Optionally pass a song name to search for lyrics for any track, even if it's not playing.
Track

Set playback volume between 0 and 100. Run with no value to check the current volume level without changing it.
TrackDJ

Jump forward by a specific number of seconds in the current track. Automatically blocked on live streams.
TrackDJ
/loop [track / queue]
Toggle loop mode. Use track to repeat the current song, or queue to loop the whole queue. Run the same option again to disable.
TrackDJ
/queue [page?]
View the server queue — 10 tracks per page with duration, title and requester for each. Shows the currently playing track at the top. Pass a page number for long queues.
Queue
/shuffle
Randomly shuffle all tracks currently in the queue without affecting the now-playing track.
QueueDJ

Move a track from one position in the queue to another. Both values must be valid positions within the current queue length.
QueueDJ

Skip everything up to a position in the queue — clears all tracks before it and starts playing from that point.

How it works
Setup & Permissions

DJ Role

Most queue and playback controls require the user to have a DJ role. Create a server role named exactly DJ and assign it to trusted members. Without it, users can still use /play, /search, /queue, /nowplaying and /lyrics freely.

Default Channel

Use /setdefaultchannel to pin all bot responses to one text channel — great for keeping your server tidy. Requires Manage Guild permission. Remove it any time with /removedefaultchannel.

Voice Channel Rules

You must be in a voice channel to use any playback commands. If the bot is already active in a different voice channel in your server, join that same channel before issuing commands.

Supported Sources

BRQXOX supports YouTube search and direct URLs, full Spotify playlist imports, SoundCloud tracks, and direct file URLs. Live streams are fully supported with 🔴 LIVE indicators in the queue and now-playing embeds.
